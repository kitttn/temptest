####AP Module Power-on Sequence

Hook the debug board up, and hit the ON\_KEY button to power the AP
Module from VCONN.

To get Unipro linkup:

1.  Remove power
2.  Hold SVC in reset
3.  Hit ON\_KEY on the debug module to turn on AP Module
4.  Release SVC reset
